# Parallel DVH Calculator & Data Generator

Проект на C++ с использованием OpenMP для параллельной генерации и анализа 3D-массивов дозиметрических данных. Программа рассчитывает гистограмму доза-объем (DVH - Dose Volume Histogram)

## Особенности
- **Параллельные вычисления:** Использование OpenMP для ускорения обработки массивов (10^6 - 10^8 элементов).
- **Эффективная память:** Пакетная генерация данных (Chunking) позволяет создавать файлы объемом больше RAM.
- **Режимы генерации:**
  1. Равномерный шум (Uniform).
  2. Нормальное распределение (Gaussian).
  3. Имитация пучка (Hotspot) — спад дозы от центра к краям.

## Структура
- `src/inputgen.cpp` — Генератор входных данных (x y z dose).
- `src/dose_par.cpp` — Основная программа анализа и построения графика.
- `data/` — Примеры входных данных.
- `screenshots/` - Примеры работы и рисунки

## Сборка и запуск

Для компиляции требуется компилятор с поддержкой OpenMP.
